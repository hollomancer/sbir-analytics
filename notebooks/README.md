# Getting Started with SBIR Analytics Notebooks

This directory contains Jupyter notebooks for interactive analysis of SBIR data, fiscal returns, and CET classification.

## Prerequisites

To run these notebooks, you need to install the ML dependencies:

```bash
make install-ml
```

## Running Notebooks

Start the Jupyter Lab server:

```bash
make notebook
```

This will open Jupyter Lab in your browser.

## Available Notebooks

- `getting_started.ipynb`: A quick introduction to loading data and running basic analysis.
